## 项目介绍
基于ssm的校园图书捐赠管理系统：SSM（Spring+SpringMVC+MyBatis）框架写的一个图书捐赠管理系统，包括用户端和管理员端。前端模板框架使用的是ace-admin，数据库使用MySQL。该系统虽然分了两个用户角色（用户端/管理员端），但是业务逻辑非常简单，还是比较适合学习SSM的同学们。

## 功能介绍

### 管理员端功能
- 查看用户列表
- 查看、修改、删除待审核的图书
- 查看待用户捐赠的图书
- 查看未通过审核的图书
- 查看捐赠记录
- 信息查看、信息修改、密码修改
### 用户端功能
- 用户登录
- 查看图书捐赠信息
- 捐赠图书
- 查看图书捐赠记录
- 查看待审核的图书
- 查看已通过审核的图书
- 查看未通过审核的图书
- 信息查看、信息修改、密码修改
